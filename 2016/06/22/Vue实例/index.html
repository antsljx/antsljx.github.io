一个 Vue 实例其实正是一个 MVVM 模式中所描述的 ViewModel - 因此习惯上会使用 vm（并不是固定的） 这个变量名。在实例化 Vue 时，需要传入一个对象，它可以包含数据、模板、挂载元素、方法、生命周期钩子等选项。全部的选项可以在 API 文档中查看。
